# ACR Documentation

## Overview

This repository contains the research and architectural documentation for ACR (Autonomous Cognitive Runtime).

ACR is a cognition-native runtime architecture research initiative focused on autonomous offline edge intelligence.

The documentation in this repository defines:
- runtime concepts
- architecture
- cognitive primitives
- execution semantics
- scheduling models
- capability abstraction
- research direction
